Hangman Game

Description

This is a beginner-friendly Hangman Game developed in Python. The program randomly selects a word from a predefined list, and the player guesses the word one letter at a time. The game ends when the player guesses the complete word or runs out of chances.

Features

- Random word selection
- Six guessing attempts
- Displays correct and incorrect guesses
- Easy to understand Python code
- Beginner-friendly project

Technologies Used

- Python 3

How to Run

1. Install Python 3.
2. Save the file as "hangman.py".
3. Open a terminal or command prompt.
4. Run the program using:
   python hangman.py
5. Start guessing letters and enjoy the game.
